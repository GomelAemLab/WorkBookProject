- sly

The SLY element does not get displayed in the resulting HTML:

<sly>This text has SLY tag in HTL</sly> >> produces >> This text has SLY tag in HTL

- unwrap

Removes the host element from the generated markup while retaining its content:

<span data-sly-unwrap="\${true}">This text has 'data-sly-unwrap' in HTL</span> >> produces >> This text has 'data-sly-unwrap' in HTL

<span data-sly-unwrap>This text has 'data-sly-unwrap' in HTL</span> >> produces >> This text has 'data-sly-unwrap' in HTL

- attribute

Adds attributes to the host element:

<span title="Lorem Ipsum" data-sly-attribute.title="This text will be instead 'Lorem ipsum' in the 'title' attribute">Inspect 'title' attribute</span> >> produces >> Inspect 'title' attribute

- text

Replaces the content of its host element with the specified text:

<span data-sly-text="This text will be instead 'Lorem ipsum'">Lorem ipsum</span> >> produces >> Lorem ipsum
